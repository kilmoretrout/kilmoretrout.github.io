popgenml

API reference.

Stats

popgenml.data.stats.sfs(x)

Compute the site frequency spectrum (SFS) from binary haplotype data.

popgenml.data.stats.theta_pi(x, pos)

Compute nucleotide diversity (π), the average number of pairwise differences per site.

popgenml.data.stats.watterson_theta(x, pos)

Compute Watterson's theta, an estimator of genetic diversity based on segregating sites.

popgenml.data.stats.tajimas_d(x, pos[, ploidy])

Compute Tajima's D, a neutrality test statistic comparing Watterson's theta and nucleotide diversity.

popgenml.data.stats.ld_stats(x[, ploidy])

Compute the mean and standard deviation of pairwise linkage disequilibrium (LD) using Rogers-Huff's r.

popgenml.data.stats.het_diversity(x[, ploidy])

Compute the mean and standard deviation of observed heterozygosity across sites.

Functions / Conversions

popgenml.data.functions.newick_to_tree(string, *)

Create a tree sequence representation of the specified newick string.

popgenml.data.functions.tree_to_graph(tree)

Convert a TSKit tree into a node feature array and edge list (graph representation).

popgenml.data.functions.graph_to_tree(x, edges)

Convert node features and edges into a TSKit tree.

popgenml.data.functions.distmat_to_tree(D[, ...])

Construct a TSKit tree from a distance matrix using hierarchical clustering.

popgenml.data.functions.tree_to_distmat(tree)

Convert a TSKit tree into a condensed genealogical distance matrix.

popgenml.data.functions.pad_sequences(sequences)

Pad a list of 2D arrays (sequences) to the same number of rows (sites) using a given padding value.

popgenml.data.functions.to_unique(X)

Identify unique site patterns across a genotype matrix and compute their frequencies.

popgenml.data.functions.seriate_spectral(x, C)

Reorder rows in a matrix using spectral seriation (Fiedler vector approach).

Relate

popgenml.data.relate.relate(X, sites, ...[, ...])

Run RELATE (a genealogy-based inference method) on simulated or empirical binary haplotype data.
